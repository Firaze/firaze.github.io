SEM Pathways

Project information

  • Category: Data mining
  • Project: Repository
  • Interface: Link to streamlit

The goal of this project was to identify the most relevant edges in a pathway.Given a pathway, the first step consisted of the extraction of all the 3-nodes cliques (triangles) with a certain configuration (for example, the first image shows what kind of triangle is needed). Then, a SEM analisys was run to estimate the weight of the edges in each clique. Finally, the edges are filtered according to a criterion. Futhermore, an interface was developed to interact with the pathways and visualize the results.
